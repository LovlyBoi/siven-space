# 智文的个人网站

## 这是个啥

这是一个 Vue3 + TS + Pinia 的项目。

这也是一个包含服务器端的项目。

服务器端使用了 express + mongoose 搭建。

这是我的第一个像样的全栈项目，Node 还没有系统学，所以后端代码可能看着比较乱，但起码功能实现了。🍻

设计参考了 https://qzhai.net/buy。
